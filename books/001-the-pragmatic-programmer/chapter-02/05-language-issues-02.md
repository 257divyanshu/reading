# 🔥 Big Insight

👉 Two types of duplication:

### 1. Language-imposed duplication

* Hard to remove
* But compiler helps catch errors

### 2. Logical/business duplication

* Dangerous
* NOT caught automatically

---

# 🔥 Key Takeaway

👉 If duplication is unavoidable:

* Prefer cases where **tools/compilers enforce consistency**

---

# 🧠 Final Mental Model

```text
Bad duplication → silent bugs (dangerous)
Language duplication → compiler catches (manageable)
Automation → reduces manual duplication (best)
```

---

# 🧾 One-line note for you:

**Some duplication is unavoidable due to language constraints, but tools and compilers help manage it by catching inconsistencies early.**
