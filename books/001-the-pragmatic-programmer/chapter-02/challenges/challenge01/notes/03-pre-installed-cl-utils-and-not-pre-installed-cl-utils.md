## 🔹 Pre-installed command-line utilities

### On Linux / macOS

* `ls`
* `grep`
* `cat`
* `sort`
* `awk`
* `sed`
* `wc`
* `head` / `tail`

### On Windows (Command Prompt / PowerShell)

* `dir`
* `find`
* `tasklist`
* `ipconfig`
* `Get-ChildItem` (PowerShell)
* `Select-String` (PowerShell)

* `docker`
