## 🔹 Pre-installed (Windows)

* `dir` → list files/directories
* `cd` → change directory (built-in)
* `copy` / `xcopy` / `robocopy` → copy files/folders
* `del` → delete files
* `type` → display file contents
* `find` → search text in files
* `tasklist` → list running processes
* `taskkill` → kill processes
* `ipconfig` → network configuration info
* `ping` → test network connectivity

### PowerShell

* `Get-ChildItem` → list files (like `ls`)
* `Set-Location` → change directory
* `Get-Content` → read file
* `Select-String` → search text (like `grep`)
* `Get-Process` → list processes
* `Stop-Process` → kill process

## 🔹 Commonly installed utilities

* `git` → version control
* `curl` → make HTTP requests
* `wget` → download files from web
* `node` / `npm` → JavaScript runtime + package manager
* `python` → scripting/programming
* `docker` → containerization
* `kubectl` → Kubernetes management
* `7z` (7-Zip CLI) → file compression/extraction
* `ffmpeg` → audio/video processing
* `ripgrep (rg)` → fast text search
* `fzf` → fuzzy finder for files/commands
