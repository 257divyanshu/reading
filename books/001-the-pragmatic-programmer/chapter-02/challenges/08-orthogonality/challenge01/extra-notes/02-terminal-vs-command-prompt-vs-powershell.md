They’re related but not the same thing:

---

## 🔹 Terminal

👉 A **program that provides a window/interface** to interact with a shell.

Examples on Windows:

* Windows Terminal
* Command Prompt window
* PowerShell window

➡️ Think: **UI container**

---

## 🔹 Command Prompt

👉 A **shell** on Windows (`cmd.exe`)

* Older, basic shell
* Uses commands like:

```bat
dir
cd
copy
```

---

## 🔹 Windows PowerShell

👉 A more advanced **shell + scripting environment**

* Object-based (not just text)
* Much more powerful than Command Prompt

Example:

```powershell
Get-ChildItem
```

---

## 🔥 Relationship

```text id="rgtfhv"
Terminal → hosts a shell
Shell    → executes commands
```

👉 On your machine:

* Windows Terminal = terminal
* Command Prompt = shell
* PowerShell = shell
