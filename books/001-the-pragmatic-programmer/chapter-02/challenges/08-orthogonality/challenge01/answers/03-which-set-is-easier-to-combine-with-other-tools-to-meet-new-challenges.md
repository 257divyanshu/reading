**The small, combinable command-line utilities are easier to combine.**

---

### 🔥 Why?

* Designed to work with **standard input/output (pipes)**
* Each tool is **independent and focused**
* Can be chained in different ways without modification

---

### 🧠 Core idea

```text
CLI utilities → composable → easy to combine for new tasks
GUI tools     → self-contained → hard to combine
```
